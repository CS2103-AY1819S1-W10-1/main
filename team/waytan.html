PROJECT: TheTracker (v1.3)

Overview

The Tracker (v1.3) is a desktop application that helps NUS students to monitor contacts, modules and events.

Summary of contributions

  • Major enhancement: improved existing delete function to delete any type of entry

    • What it does: allows the user to delete a person, module or occasion by index

    • Justification: This feature is necessary for a user to simultaneously maintain all three types of entries.

    • Highlights: This enhancement required the different lists to be handled independently, but with all actions tracked simultaneously (e.g. for undo/redo).

Contributions to the User Guide

Given below are sections I contributed to the User Guide. They showcase my ability to write documentation targeting end-users.

Deleting an entry : delete

Deletes the specified person, module or occasion in TheTracker.

Deleting a person : deleteperson

Format:
deleteperson INDEX

Examples:

  • deleteperson 2
    Deletes the 2nd person.

Deleting a module : deleteModule

Format:
deletemodule INDEX

Examples:

  • deletemodule 1
    Deletes the 1st module.

Deleting an occasion : deleteOccasion

Format:
deleteoccasion INDEX

Examples:

  • deleteoccasion 5
    Deletes the 5th occasion.
